About SeaGlide

SeaGlide is built for students and combines elements like research and engineering to design an autonomous drone. This drone is used to tackle real-world problems and it does this by collecting varies data as it drifts throughout the water.

Arduino Programming

The underwater drones are programmed using Arduino. Arduino is an open-source electronics software and hardware that takes in inputs and turns it into outputs. For example: light hits a sensor, it activates a motor. Arduino References
